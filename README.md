# Bike Rental Website

Welcome to the Bike Rental Website project! This React-based web application allows users to browse and rent bikes online.

## Getting Started

### Prerequisites
Before you begin, ensure you have the following installed on your machine:

- [Node.js](https://nodejs.org/en/): This project requires Node.js to be installed. You can download and install it from the official website.

### Installation
1. Clone the repository to your local machine using the following command:

   ```
   git clone https://github.com/ChristopherMichaelShea/bike-rental-website.git
   ```
   
2. Navigate to the project directory:

    ```
    cd bike-rental-website
    ```
3. Install the project dependencies using npm (Node Package Manager):

   ```
   npm install
   ```
   
### Running the Application
Once the installation is complete, you can run the application with the following command:

```
npm start
```

This command starts the development server and opens the application in your default web browser. If it doesn't open automatically, you can access it by navigating to http://localhost:3000 in your browser.

### Features
Bike Catalog: Browse a catalog of available bikes with details such as type, size, and rental price.
Booking System: Select a bike, choose rental dates, and proceed to book bikes for a specific period.
